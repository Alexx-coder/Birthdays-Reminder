# Relatives Birthdays

Relatives Birthdays - Программа которая локально хранит данныее о днях рождениях ваших родственниках

## Прошлая версия

- [Прошлая версия v1.0](https://github.com/Alexx-coder/birthdays_relatives.git)

## Что нового добавили в эту версию

- База данных SQLite3

- Улучшенные старые функции

- Регистрация / Вход

- Безопасность

### Улучшенные функции

#### Создание информации о родственнике:

- Бесконечное количество родственников (Было макс. количество - 8 родственников)

- Происходит коннект с базой данных 

- Проверка даты (Формат)

#### Просмотр информации о родственнике

- Происходит коннект с базой данных

### Безопасность

- Хэширование пароля
